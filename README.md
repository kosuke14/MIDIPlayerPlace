# MIDIPlayerPlace
MIDI Player Place File for Roblox, and MIDI Examples.<br/>
This is a MIDI Player in Roblox.<br/>
Not a Piano Player.

## Features
- Originally implemented synthesizer
- works like synthesia
- open sourced (.rbxl file)
- it can decode midi in roblox
- almost full instruments (+ earthbound)
